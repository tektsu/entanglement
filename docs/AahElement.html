Class: AahElement

AahElement(g, center, options)

Define the Aah element. An Aah element is star-shaped. It needs an approximate size and a position. The size is approximate because the aah's components are built with some size variations. In addition, an Aah can be rotated randomly, and the angle between the arms can vary.

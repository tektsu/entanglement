Constructor

new Zentangle(size, shape, options)

Create a Zentangle object.

Parameters:
Name Type Description
size number

The size of the Zentangle in pixels. There is only one number, as Zentagles are a square or a triangle (in which case size is the length of the side), or a circle (in which case the size is the diameter.)
shape string

The shape of the Zentangle. Can be 'square' (the default), 'triangle' or 'circle'.
options ZentangleOptions

Class: BoxSpiralElement

BoxSpiralElement(g, center, options)

Define the BoxSpiral Element. The BoxSpiral element is a square spiral which is used in several tangles. Its size and direction of rotation (cw or ccw) can be specified in the options. The spiral can be made to fit any quarilateral; it need not be limited to a square.
